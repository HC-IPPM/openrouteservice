Isochrones

Isochrones represent areas that can be reached within a certain time from a starting point, given a specific mode of transportation.

Routing

Routing calculates the best route between a starting point and a destination, using a specified mode of transportation.

Matrix

Matrix provides travel distances and times between multiple points, useful for analyzing multiple routes and locations.

Snap

Snap adjusts locations to the nearest roads or pathways, useful for aligning points to the transportation network.
